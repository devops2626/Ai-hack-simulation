# Contributing to AI Hacking Simulator

Thank you for considering contributing to this educational project!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run the full CI suite locally if possible
5. Submit a Pull Request

## Development Setup

```bash
make install
make run-agent
```

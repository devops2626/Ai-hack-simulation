# Security Policy

## Supported Versions

This is an **educational simulation** project. Only the latest version is supported.

## Reporting a Vulnerability

If you discover a security vulnerability in this simulation code:

1. Please report it responsibly via GitHub Issues or email the maintainer.
2. **This is NOT a real-world exploitable system** — it's a sandbox for defensive learning.

## Disclaimer

- The vulnerable Flask server and payloads are for **educational purposes only**.
- Do not use against production systems without authorization.
- The project emphasizes ethical hacking and defensive security research.
